   - [ ] Метрики
   - [ ] Логи
   - [ ] Трейсы

    - [ ] Игнорирование ошибок для ускорения разработки
    - [ ] Откладывание исправления ошибок на потом
    - [ ] Быстрое реагирование на появление ошибок в production

11. Что НЕ является ключевой метрикой DORA для оценки DevOps?

12. Что такое Test Pyramid в стратегии тестирования?
    - [x] Много unit-тестов в основании, меньше integration тестов, минимум e2e тестов
    - [ ] Использование пирамидальной схемы для организации integration тестов

    - [ ] Минимум unit-тестов, больше e2e-тестов, максимум integration тестов в основании

13. Что такое Trunk-Based Development?

    - [ ] Использование отдельной ветки для каждого разработчика

    - [ ] Резервное копирование перед развертыванием

15. Что такое Canary Releases?

    - [ ] Откат всех изменений при первой ошибке
    - [ ] Тестирование на птицах перед развертыванием
